Bienvenue sur le site du Projet IRVE

Voici une analyse des bornes de recharge pour les voitures électriques dans les communes de Pont l'Abbé dans le Finistère et à Nanteuil-le-Haudouin, dans l'Oise. Selon les data données, il a été possible de constater que sur ces espaces, il était possible de trouver 8 bornes de recharge. Toutefois, elles se trouvent toutes à Pont l'Abbé.

Utilisez la navigation en haut pour découvrir la problématique étudiée, explorer les données utilisées, et visualiser notre carte interactive.
